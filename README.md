
#### NFT.StorageLite APP

> 基于https://nft.storage API，使用ReactNative与Expo开发的演示APP，支持Android与iOS端。

1.支持设置API KEY后查询该帐户下的所属文件列表与详情。

2.支持通过指定CID查询特定文件详情。
